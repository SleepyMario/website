My cover pictures are (obviously) done using AI. The thing actually has a sense of humour it seems, but anyway:

I think it's done. The rudimentary framework of something new is in place. This thing has the potential to completely overhaul the education system, and trust me: I won't be crying about that. The fans have heard a few horror stories so to say. 

Trust me, the real scary part is that there are many more of these completely messed up stories. 

It was a bit of a difficult decision on how to do it licensing wise. I've decided to just use gpl-3 for all the technical components; it's not a hugely complicated thing anyway. For now I put the language parts of it as available for non-commercial use though. Hey if I had a real income I might have put it out there for completely free, but I think I'll just work on it and then you can buy chapters for really cheap. The written part is textbook style, and is not free to produce, so yeah. I'm losing both time and bucks on this. Money wise this is a bit of a gamble. 

I can see how this can change education. There are a few things that need to be discussed for real, such as how to deal with different levels from different ages and maybe seriously think on how to organize hobby/sport/dancing/whatever-exactly groups and things like that. All these things serve a function, some are very innocent of course, but some are innocent but actually important. 

Mon-Fri 8-5 school isn't necessarily the most important thing on earth anymore. I'm sure these things are changing over the years; I was asking this bot thing about certain curricula in certain places and was surprised; Actually things seem to have changed quite a fair amount already. I'm pretty sure not in universities though. 

Anyway.

This is not a replacement for a real language curriculum by the way. I'll write some more on this on the whacksmacker website itself, but it's not designed as a standalone. You can learn some things quickly though using this, but you still need to practice listening, speaking, singing, etc, etc, etc. It's the dry technical part so to say, not the real life part. 

Both are important.

Curriculum wise today we will finish up to ch 140 for Dutch. Tomorrow I will restart working out and studying Vietnamese. I had to put it on hold for both mental and physical reasons. 

Things seem okay now. They're not okay. But I'm still alive. 

Not that I like it, but I mean, I'm alive, so yeah. 

I guess I'll have to live with it. 

Right?
